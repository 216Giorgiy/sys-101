Exercices

  1. What time is it?
    1. Write a script that outputs "Good morning", "Lunch time!", "Good afternoon", "Good evening" or "Go to sleep!" according to the hour.
    2. How to add this as the start of every prompt.
  2. Using the log files provided:
    1. Use wget to download http://sys-101.auzias.net/practice/log01
    2. Use curl to download http://sys-101.auzias.net/practice/log02
    3. How many lines do these two files have?
    4. Create one big file (merge them).
    5. How many lines does the big file have?
    6. Sort them chronologically.
    7. Show 7 lines preceding the line #63 (included). What did happen?
    8. How many different log types are in this file? The output should be shown by `wc`.
    9. Display the different log types only.
    10. Remove the '%'.
    11. How much does the file weight?
    12. How much does the file really weight?
    13. Make it a real csv file.
    14. Remove unnecessary WS. Does the file weight smaller?
    15. The file has the form "k=v", remove "k=" to just keep the values. Does the file weight smaller? Need help? Don't know `man`?
    16. When was the battery at its lowest? Display the whole line. Do not use `cut`.
    17. What was the temperature at its highest?
  3. All rights!
    1. Check the permissions of your homedir
    2. Check if a user allow you to list its homedir
  4. System call
    1. What does the command `strace` do?
    2. Assuming that one output line of `strace` is one system call:
      1. Save the output of the command you are about to perform to process them later
      2. Does the number of system calls differ to copy a big or a small file? What about mv or cat?
      3. What is the command with the lowest, and the one with the biggest, number of system call between: rm, cp, touch, ls, mv, cat?
      4. During the lesson several system calls were presented. What are the most system calls performed for each command?
      5. How many write are performed per kB written when using `dd`?
      The command "dd if=/dev/zero of=/tmp/rm bs=8k count=2000 && sync" could help you.
  5. How to remove the file named "-rf"?
