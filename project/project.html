Project ZombiLand

Ce projet a été proposé et organisé par les élèves. Cette page est une version corrigée de leur rédaction pour présenter le sujet.

Présentation

Présentation générale

ZombiLand est un projet de simulation d'externimasion de [processus] zombies sur votre PC.
Le principe est simple une vague de zombies est créé par le programme MaitreZombie.c et vous devrez défendre votre système d'exploitation en tuant ces zombies. Trois armes différentes à votre disposition sous forme de programme C.

Présentation technique

Les processus Zombies sont des processus déjà terminés dont les parents attendent toujours un changement d'état. Étant terminé, un processus zombie ne peut être tué via des envoies de signaux et ils ne sont plus tuables. Pour nettoyer un processus zombie il faut donc tuer son parent.

En C pour trouver les processus Zombie et parent, la fonction System PROC vous sera utile. De plus les Parents de Zombie ne sont tuables que d'une certaine façon :

  1. Zombies avec PID de 0 mod 3 sont vulnerables aux handguns
  2. Zombies avec PID de 1 mod 3 sont vulnerables aux knives
  3. Zombies avec PID de 2 mod 3 sont vulnerables aux shotguns

      4. Il est possible de tuer tous les zombies d'un coup en envoyant un signal INT au binaire MaitreZombie.

      Ce qui est fourni

      Pour ce faire, vous avez à disposition le programme qui crée les zombies, ainsi que les différentes armes. Programmes à jours disponibles ici.

      Travail à faire

      Vous devez donc créer un programme tuant les parents des zombies. Vous devrez pour cela:

      1. Trouver les zombies: identification des processus en état "zombie".
      2. Déterminer quel est le parent de chacun de ces zombies.
      3. Tuer ce parent avec l'arme correspondant au PID.

          4. Ce projet sous Linux et en C est possible en lisant les fichiers /proc/[1-9]*/status pour trouver les processus zombies et un getppid() pour trouver le processus parent. Cf `$man proc` et `$man getppid`.
          Une version améliorée d'une version avec un seul thread tuant les zombies pourrait être d'avoir:

          • Un thread pour chercher les zombies.
          • Un thread pour tuer les zombies (ou trois, un thread par arme).
          • Un thread vérifiant la création de nouveaux zombies.
