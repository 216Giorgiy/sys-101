Présentation

Présentation générale

ZombiLand est un projet de simulation de l'apocalypse sur votre PC. Le principe est simple une vague de zombies veut un smoothie de cerveau et vous êtes le seul fournisseur. Votre seul moyen de survie est donc de tuer tous ces zombies avec les trois armes à votre disposition. D'un point de vue informatique cela donne donc des processus Zombies et un programme en C qui modélise votre survivant.

Présentation technique

Les processus Zombies sont des processus déjà morts mais dont les parents attendent toujours quelque chose. Comme ils sont morts ,ils ne sont plus tuables. Pour terminer un processus zombie il faut tuer son parent.

En C pour trouver les processus Zombie et parent, la fonction System PROC vous sera utile. De plus les Parents de Zombie ne sont tuables que d'une certaine façon :
Zombies avec PID de 0 mod 3 sont vulnerables aux handguns
Zombies avec PID de 1 mod 3 sont vulnerables aux knives
Zombies avec PID de 2 mod 3 sont vulnerables aux shotguns

Ce qui est fourni

Pour ce faire, vous avez de fourni le programme qui crée les zombies, ainsi que les différentes armes. Ici le git

Travail à faire

Vous devez donc créer le programme du survivant en C. Vous devez eliminer au mieux les zombies, simplement cela. Une fois cela fait restez branchés d'autres niveaux du programme sortiront.
